# Ludo Championship

A polished, browser-based Ludo game built with HTML, CSS, and JavaScript. This project recreates the classic board game in a modern, colorful interface with animated dice, token movement, captures, and a celebratory win screen.

![Ludo Championship](https://img.shields.io/badge/Status-Playable-brightgreen) ![HTML](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white) ![CSS](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

## ✨ Features

- 8-player Ludo gameplay
- Interactive board with animated dice rolling
- Turn-based token movement and capture logic
- Home stretch and victory flow
- Neumorphic UI design with vibrant player colors
- Simple, lightweight, and easy to run locally

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ludo-championship.git
   ```
2. Open the project folder.
3. Start the game by opening [index.html](index.html) in your browser.

If you prefer a local server:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## 📁 Project Structure

- [index.html](index.html) — Main HTML structure and game UI
- [style.css](style.css) — Game styling and visual design
- [logic.js](logic.js) — Core game logic and board interactions

## 🎮 How to Play

- Click the Roll Dice button to begin your turn.
- Roll a 6 to bring a token into play.
- Click a glowing token to move it across the board.
- Capture opponent tokens when they land on the same position.
- Reach the home area to win the round.
